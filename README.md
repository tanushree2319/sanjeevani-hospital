# Sanjeevani Hospital Website

A responsive multi-page hospital website built using HTML, CSS, and JavaScript, focusing on clear UI structure, usability, and real-world functionality.

## Features

- Multi-page layout (Home, About, Services, Contact)
- Responsive design using Bootstrap
- Hero image slider with smooth transitions
- Scroll-based animations using Intersection Observer
- Appointment form with WhatsApp integration
- Floating contact actions (call and WhatsApp)
- Clean typography and consistent UI styling

## Approach

This project focuses on structuring a real-world service website with attention to layout clarity, accessibility basics, and user interaction without relying on heavy frameworks.

## Tech Stack

- HTML5
- CSS3 (custom styling + variables)
- JavaScript (DOM interactions)
- Bootstrap

## Purpose

Built to explore how clean UI structure and lightweight interactions can create a functional and user-friendly experience for a real-world healthcare website.
